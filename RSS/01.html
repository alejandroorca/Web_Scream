1. ¿Qué es la sindicación web? ¿Qué son las fuentes web? ¿Qué relación tienen con la tecnología XML?

  1. Sindicación Web: es el reenvío o reemisión de contenidos desde una fuente original o sitio web de origen hasta otro sitio web de destino que actúa como receptor y que a su vez se convierte en emisor puesto que pone los contenidos a disposición de sus diferentes tipos de usuarios; la Sindicación Web se lleva a cabo mediante un contrato o licencia entre las partes, como por ejemplo, sitio web de origen y sitio web de destino.
  2. Fuentes web: es un medio de redifusión de contenido web. Se utiliza para suministrar información actualizada frecuentemente a sus suscriptores. En su jerga, cuando una página web redifunde su contenido mediante una fuente web, los internautas pueden "suscribirse" a ella para estar informados de sus novedades. Los interesados pueden usar un programa "agregador" para acceder a sus fuentes suscritas desde un mismo lugar.
  3. Relación con tecnología XML:Se usa la tecnología XML.
